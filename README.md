# README

### Nimbl3 Scraper

A web​ application that​ will​ extract large amounts of​ data​ from​ the​ Google​ search results​ page. 

### Prerequisites
 
* Ruby version: 2.4.2

* Rails version: 5.1.4

* Database: PostgresSQL

* Front-end Framework: Twitter Bootstrap 4

* URL: https://guarded-fortress-72816.herokuapp.com/

### CSV Examples

```
id,term
1,lazada
2,11street
3,samsung
4,apple
5,dtac

```
